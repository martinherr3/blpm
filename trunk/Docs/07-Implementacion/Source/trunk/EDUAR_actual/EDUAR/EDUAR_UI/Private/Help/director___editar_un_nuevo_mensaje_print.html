Director - Editar un nuevo mensaje

 
 
Para el caso de un docente, se brinda la posibilidad de enviar mensajes a:
  • Destinatarios sin restricción de curso o tipo de destinatario (si no selecciona la opción de filtrado por cursos)
  • Determinados alumnos de un curso dado, seleccionando uno a uno
  • Todos los alumnos de un curso dado, seleccionando la opción "Alumnos de XX Año Y"
  • Determinados tutores de un curso dado, seleccionando uno a uno
  • Todos los tutores de un curso dado, seleccionando la opción "Tutores de XX Año Y"
  • Determinados docentes de un curso dado, seleccionando uno a uno
  • Todos los docentes de un curso dado, seleccionando la opción "Docentes XX Año Y"
 

Enviar el mensaje

Envia el mensaje a los destinatarios seleccionados, se obtiene una advertencia si no se selecciona un destinatario

Volver

Vuelve a la página anterior sin enviar el mensaje

Opción de filtrado por curso

Ya sea que el docente pertenezca a un solo curso como a varios, la selección es obligatoria a fin de establecer los destinatarios posibles.
 
Cuando se selecciona dicha opción se despliegan mas opciones para el curso y tipo de destinatario
 

Selección de los destinatarios

En base al curso y tipo de destinatario, se listan los posibles destinatarios, pudiendo seleccionar uno, un grupo o bien todos los destinatarios listados.

Asunto del mensaje

Es de importancia a fin de hacer saber al destinatario el propósito del mensaje, si bien no es obligatorio el sistema solicita la confirmación para enviar el mensaje sin un asunto.
 

Opciones de formato de mensaje

Se brinda un amplio espectro de opciones para brindar al usuario la capacidad de editar su mensaje con formato
