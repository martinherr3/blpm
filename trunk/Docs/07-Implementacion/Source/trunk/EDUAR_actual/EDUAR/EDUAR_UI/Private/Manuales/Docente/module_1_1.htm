Para Acceder al Sistema

Para acceder al sistema es necesario identificar al usuario. Por esto es necesario ingresar usuario y contraseña.

Nombre de Usuario

 
En este campo debe ingresar el nombre del usuario.
 
[Arriba]

Contraseña

En este campo se debe ingresar la contraseña.
[Arriba]

Botón Iniciar sesión

Hacer clic en este boton para dar inicio a su sesion.
[Arriba]

Registrar nuevo Usuario

Si es la primera vez que ingresa al sistema debe Registrarse. Hacer Clik en Registrarse para completar datos requeridos de su nueva cuenta en Eduardo y obtener su nombre de usuario y contraseña. En la sesion de "La primera vez que ingreso a Edu@r2.0" de este manual le mostrara que datos ingresar para la registracion.
[Arriba]

Olvido de Contraseña

En caso de que no recuerde su contraseña hacer clik en el link Aquí para proceder a la obtencion de su contraseña olvidada.
[Arriba]
